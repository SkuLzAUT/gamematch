Der Schutz deiner persönlichen Daten ist uns ein besonderes Anliegen. Diese Website verarbeitet Daten ausschließlich auf Grundlage der gesetzlichen Bestimmungen (DSGVO, TKG).

Verwendete Dienste:

  • Firebase / Google Cloud: Wird für Authentifizierung, Datenbank, Hosting und ggf. Analytics verwendet. Es können personenbezogene Daten (wie E-Mail-Adresse) verarbeitet werden.
  • Google Fonts: Schriftarten können von Google-Servern geladen werden. Dabei wird deine IP-Adresse an Google übermittelt.
